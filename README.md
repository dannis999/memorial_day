# memorial_day
纪念日计算器，可用于提醒自己恋爱纪念日等特殊日期

数据文件：datas.txt（若该文件不存在，程序会自动读取datas_test.txt）

- 可仿照datas_test.txt进行准备
- 一行一条数据，使用逗号分隔，分别表示 年、月、日、说明、注释
- 例如：`2020,10,10,和github在一起,`
